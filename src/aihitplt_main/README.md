# 功能包使用说明

## 1. 功能包说明

`aihitplt_main` 功能包内包含机械臂、AGV运动控制、送物上装、安放上装等功能API程序，给用户提供了一个统一的接口来控制AIHIT系统的各个功能模块，方便用户进行开发和调试。同时功能包内还包含了应用场景程序，用户可以通过运行这些程序来体验AIHIT系统在不同场景下的表现。

## 2. 功能包使用

`rosrun aihiplt_main security_main.py` # 启动安防场景程序界面

`rosrun aihitplt_main main.py` # 运行API功能主程序，通过在该文件中编写或调用API函数来实现对AIHIT系统的控制和功能测试

## 3. API说明

AIHIT系统提供了一系列API函数，用户可以通过调用这些函数来实现对机械臂、AGV运动控制、送物上装、安放上装等功能的控制。具体的API函数说明和使用方法可以参考功能包内的API文档或示例代码。
