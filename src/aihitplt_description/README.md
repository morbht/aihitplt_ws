# 功能包使用说明

## 1. 功能包说明

`aihitplt_description` 功能包用于存储机器人的URDF模型和相关的描述文件

## 2. 功能包使用

`roslaunch aihitplt_description display.launch` # 启动RViz显示机器人模型

`roslaunch aihitplt_description display_gripper.launch` # 启动RViz显示夹爪模型

`roslaunch aihitplt_description display_with_arm.launch` # 启动RViz显示机器人及机械臂模型

`roslaunch aihitplt_description gazebo.launch` # 启动Gazebo仿真环境显示机器人模型

`roslaunch aihitplt_description gazebo_with_arm.launch` # 启动Gazebo仿真环境显示机器人及机械臂模型
